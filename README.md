# Fuzzy-ML Blood Supply Chain (Python Code)

This repository contains the Python implementation of the models proposed in the paper:

**Optimizing a Multi-Objective Fuzzy Mathematical Model to Predict the Donable Blood in a Blood Transfusion Supply Chain Using Machine Learning Algorithms Considering Disruption**

## Overview
The code implements a multi-objective fuzzy mathematical model combined with machine learning algorithms to predict donable blood and optimize decision-making in a blood transfusion supply chain under disruption scenarios.
